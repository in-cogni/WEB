DOM - Document Object Model

DOM представляет любую HTML-страницу как сложный объект, который агрегирует в себя другие объекты — элементы страницы. DOM объединяет все элементы страницы в дерево объектов, корневым объектом которого является объект «document». Этот объект всегда неявно создается в любом JavaScript-коде.
